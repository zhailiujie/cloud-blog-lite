# 运维说明

## 备份

- Worker 定时任务会执行 D1 备份并写入 R2。
- 备份对象位于 `backups/d1/daily/`。
- 每次备份后会清理 3 天前的备份对象。
- 备份内容包含核心数据表，不包含操作日志。

## 操作日志

操作日志记录关键后台操作，包括：

- 站点创建、更新、删除
- 站点导入、导出
- 标签创建、更新、删除
- 用户管理
- 系统设置更新
- 日志清理

后台日志页支持按模块、动作、用户名筛选，并可查看 JSON detail。

日志保留策略：

- Cron 每天执行一次，自动删除 7 天前的操作日志。
- 后台日志页的手动清理按钮同样清理 7 天前的操作日志。
